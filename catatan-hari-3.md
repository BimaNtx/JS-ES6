## EPS 5 map, filter dan reduce

map() = Melakukaan transformasi pada setiap elemen array dan menyimpannya di array baru 
filter() = Menelusuri array dan mengembalikan elemen yang memenuhi kondisi tertentu(bernilai true)
reduce() = Menelusuri seluruh array dan merangkum seluruh isinya menjadi satu nilai tunggal
